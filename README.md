## Modular 9-2019

# EOS Named Account Creation

Creates an eos account name via a frontend



## 1 | Getting Started


### 1.1 Installation

Clone, then install dependencies in the server and client.

```
cd server
npm install
```

```
cd client
npm install
```

### 1.2 Running

Run the server first, then the client.

Sever:

```
npm run dev
```

Web:

```
npm run web
```

Mobile:

```
npm run start
```
